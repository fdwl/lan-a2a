# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-01-01

### Added

- Initial release
- A2A protocol adapter with AgentCard, Task, Message support
- P2P transport with mDNS discovery
- WebSocket-based communication
- Relay server for cross-subnet scenarios
- Channel management (P2P Lobby and Relay modes)
- File transfer with chunked concurrent transfers
- Folder sync with incremental updates
- MCP JSON-RPC server with 11 tools
- Plugin system with event hooks
- Agent profile management
- TypeScript client (basic)
- Docker support (multi-stage build)
- OpenAPI specification
- Wire protocol documentation
